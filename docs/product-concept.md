## 2. コンセプト

### 一言で

**記憶力の低下に悩む大人が、自分の人生の思い出で遊び、気づきと笑いを得ながら、結果的に脳をやさしく活性化するアプリ。**

### ポジショニング

```
[一般的な脳トレ]          [本アプリ]              [日記・回想療法]
  課題→正解→スコア    問い→思い出す→発見     自由記述→内省
  競争・数値            あそび・共有            重い・継続しにくい
```

本アプリは **真ん中の「軽い回想 × あそびの仕組み」** を取る。

### ユーザーが感じるべきこと

| 感じてほしい | 感じさせない |
|---|---|
| 「懐かしいな」 | 「頭が悪くなった」 |
| 「そういえばあったな」 | 「正解できなかった」 |
| 「笑える・しんみりする」 | 「テストされた」 |
| 「明日も開けたい」 | 「義務・宿題」 |
| 「誰かに話したい」 | 「負けた・遅れている」 |

### コアバリュー

1. **自分の人生がコンテンツ** — 問題集ではなく、あなたの記憶が主役
2. **正解の多様性** — 違う答えが宝物
3. **5分の儀式** — 宝箱を1つ開けるだけ
4. **感覚と感情** — 頭だけでなく、体・心の記憶
5. **あそびの継続** — 義務感ではなく、コレクションと物語で戻ってくる

## 3. コア体験

### 毎日の5分ループ（MVPの心臓部）

```
┌─────────────────────────────────────────┐
│  ① 今日の「宝箱」が届く（通知 or 起動）    │
│         ↓                               │
│  ② 開封アニメ（紙袋・木箱・封筒など）      │
│         ↓                               │
│  ③ 今日の「思い出しの問い」1〜3問         │
│     （テキスト入力 or 選択 or 音声メモ）    │
│         ↓                               │
│  ④ 振り返り画面                          │
│     「あなたの答え」＋ 気づきの一言        │
│     （正誤判定なし）                       │
│         ↓                               │
│  ⑤ 報酬                                  │
│     スタンプ / しおり / キャラの一言       │
│         ↓                               │
│  ⑥ （任意）ひとこと共有 or 自分用メモ保存   │
└─────────────────────────────────────────┘
```

### 体験設計の原則

**問いの書き方**

- ❌「次のうち正しいのはどれ？」
- ✅「小学校の帰り道で、よく見ていたものを3つ思い出してみてください」

**フィードバック**

- ❌「3問中2問正解！レベルアップ」
- ✅「3つのうち、『工場の煙』を思い出したのですね。あなただけの景色ですね」

**失敗の再定義**

- 思い出せない → 「今日は出てこなかった記憶。宝箱に『？』のしおりを入れておきましょう。また会いに来ます」

### 週次・月次のコア体験（Phase 2以降）

- **週末**：今週集めた思い出しおりを「アルバム」に綴じる
- **月次**：「あの月の自分」レター（アプリが問いをまとめて振り返り）
- **季節**：季節の感覚テーマ（夏の匂い、冬の音）

## 4.5 認知刺激の3層モデル（科学設計）

> **思想はそのまま**：脳トレ感・点数・正解一つ — これらは出さない。  
> **問いの中身だけ**、科学的に意味のある認知負荷を段階的に混ぜる。

### なぜ3層か

| 層 | 主に使う脳 | 疲れ方 | 科学的意味 |
|---|---|---|---|
| **① あたたかい想起** | 海馬・感情・イメージ | ほとんど疲れない | エピソード記憶・想起練習 |
| **② ちょい深度** | 海馬＋前頭前野（軽く） | ほんの少し | 順序・分類・比較・連想 |
| **③ 深い宝箱（週1）** | 前頭前野・注意ネットワーク | 脳トレより軽い | 切り替え・二重課題・カテゴリ流暢性 |

一般的な脳トレの「疲れ」は、**前頭前野に強い負荷**（処理速度・正解競争）から来る。  
本アプリはそれを毎日やらず、**思い出しの中に自然に混ぜる**。

### 研究上の根拠（要約）

- **エピソード記憶の想起** … 海馬を使う。想起の努力（retrieval effort）自体が記憶を強化する
- **意味記憶・連想** … カテゴリ流暢性、意味ネットワークの活性化
- **実行機能** … 順番整理・分類・共通点探索（計算ドリル以外でも鍛えられる）
- **多感覚結合** … 匂い＋音＋場所など、エピソード再構成
- **認知柔軟性** … テーマの切り替え、異なる記憶間の関連づけ
- **継続** … FINGER試験等が示すように、単一課題より「複合・継続」が重要

### 配信ルール（実装済み）

| タイミング | 問いの構成 |
|---|---|
| **平日（月〜金・日）** | 深度① ×1 ＋ 深度② ×1 |
| **土曜（深い宝箱）** | 深度③ ×1 ＋ 深度② ×1 |

- 直近7日間は同じ問いを出にくい
- 深度③は5問プールからローテーション
- UIでは「深い宝箱」バッジのみ。スコア・正答率は出さない

### UIでの見せ方

| 要素 | 見せ方 |
|---|---|
| 深度② | 「ちょい深い問い」ラベル（控えめ） |
| 深度③ | 「深い宝箱の問い」＋ 宝箱に金の輪 |
| しおり | 深度②以上は箔押し風（太いしおり） |
| モチ | 「今日は少し深い箱だったね」（正誤は言わない） |

脳トレの「ストリーク地獄」ではなく、**戻ってきたくなる理由**を設計する。

### 5-1. 宝箱とコレクション

| 要素 | 説明 | 継続動機 |
|---|---|---|
| **日替わり宝箱** | 毎日違う見た目の箱（木・布・紙・鉄） | 開ける楽しみ |
| **思い出しおり** | 答えた問いがしおりになる。色は感情で変化 | 集める快感 |
| **？しおり** | 思い出せなかった問い。後日「再会」イベント | 失敗の保存 |
| **季節の箔押し** | 春夏秋冬でレアしおり | 期間限定 |

### 5-2. ガチャ（※課金なしでも回せる設計）

- **通貨**：「なつかし石」— 1日1回の宝箱完了で1個
- **ガチャ内容**：しおり枠・宝箱スキン・キャラのセリフ・BGM・問いの「種」
- **思想**：ガチャで **強さ** は出ない。あくまで **見た目と物語**
- **天井**：30回でレア確定（課金圧をかけない）

### 5-3. キャラクターとストーリー

- 案は [§7 キャラクター案](#7-キャラクター案) 参照
- キャラは **先生ではなく旅の仲間**。正解を教えない
- ストーリー：「記憶の attic（屋根裏）」を一緒に片付けていく物語
- 章進行：しおりの枚数で新エリア解放（帰り道エリア、海エリア…）

### 5-4. 共有（競争ではない）

- **匿名の思い出棚**：「帰り道に工場の煙を見ていた人、他に◯人」
- **共感ボタン**：「わかる」「いい話」「うちも」— コメント不要
- **ペアモード**（Phase 2）：家族・パートナーと同じ問いに答える

### 5-5. やさしい習慣化

| 機能 | 脳トレ的 | 本アプリ的 |
|---|---|---|
| 連続記録 | 「7日連続！途切れたらリセット」 | 「今月◯日開けた箱」カレンダー |
| リマインド | 「今日やっていません」 | 「今日の宝箱、そろそろ届きますよ」 |
| 休み | 罪悪感 | 「お休みシール」— 休んでもコレクションに残る |

## 6. UIイメージ

### デザインキーワード

**紙・木・夕暮れ・手帳・宝箱** — デジタルだが、アナログの温かさ

### カラーパレット

| 用途 | 色 | 意味 |
|---|---|---|
| 背景 | `#F5F0E8` 生成り | 古い手帳の紙 |
| メイン | `#C4A882` 薄い木 | 宝箱・棚 |
| アクセント | `#E8A87C` 夕暮れオレンジ | 発見・温かさ |
| テキスト | `#3D3630` 墨色 | 読みやすさ |
| サブ | `#8B9E8B` 薄い緑 | 安心・自然 |

### タイポグラフィ

- 見出し：丸みのあるゴシック（例：M PLUS Rounded 1c）
- 本文：読みやすい明朝 or ゴシック（例：Noto Sans JP）
- 数字・スコア：**極力使わない**。日付・しおりの枚数程度

### 主要画面の雰囲気

**ホーム（宝箱のある部屋）**

```
┌──────────────────────────────┐
│  〔柔らかい照明の「部屋」背景〕    │
│                              │
│      📦 今日の宝箱              │
│      「さあ、開けてみましょう」    │
│                              │
│  ── しおり棚 ──────────────  │
│  [赤][青][？][金] ...         │
│                              │
│  🐾 キャラ「今日は帰り道の話」   │
└──────────────────────────────┘
```

**問い画面**

- 1問1画面。スクロール最小
- 大きな問いテキスト＋小さなヒント（「思い出せなくても大丈夫です」）
- 入力欄はノート風の罫線
- 音声入力ボタン（話した方が思い出しやすい人向け）

**振り返り画面**

- ユーザーの回答を「しおり」に印刷するアニメ
- キャラの一言（共感・発見を促す）
- **正誤・点数なし**

**アルバム**

- 集めたしおりが本棚に並ぶ
- タップでその日の問いと答えを見返せる

### NGデザイン

- 脳のイラスト・メダル・ランキング表
- 赤×の不正解表示
- ゲーム的なHPバー・レベル
- 派手なネオン・eスポーツ感

### OKデザイン

- 手書き風アイコン・紙の質感
- 開封アニメ・ページをめくる音
- 空のしおり（まだ書いていない自分への余白）

## 7. キャラクター案

### メインキャラ：「モチ」

| 項目 | 内容 |
|---|---|
| 種族 | 記憶の屋根裏に住む、ふわふわの小さな生き物（犬でも猫でもない） |

| **しおり職人オバケ** | コレクションの解説 | 「この赤いしおり、夏の記憶がにじんでるね」 |
| **？のねこ** | 思い出せなかった問いの再会 | 「あの？、また会いに来たよ」 |
| **夕焼けおじいちゃん** | 感覚記憶テーマの案内 | 「匂いはね、記憶の近道なんじゃよ」 |
| **連想ツバメ** | 連想ゲームのお題を運ぶ | 「次は『雨』を持ってきたよ」 |

### キャラクター設計の原則

- **教師・医者・コーチにしない** — 上から目線を避ける
- **ユーザーの答えを否定しない**
- **年齢を問わない** — キャラも大人も子どももいない中間の世界

## 8. マネタイズ案

### 思想

課金で **記憶力が上がる** わけではない。 **体験が豊かになる** 課金。

### 案一覧

| モデル | 内容 | 優先度 |
|---|---|---|
| **無料＋プレミアム** | 月額480〜780円。広告なし・しおり枠追加・ペアモード・音声エクスポート | ★★★ MVP後 |
| **買い切りスキン** | 宝箱デザイン・部屋の装飾・しおりの箔 | ★★☆ |
| **物理グッズ** | しおりを印刷した「思い出アルバム」郵送、もなかの団体商品と連動 | ★★☆ もなからしい |
| **法人・施設** | デイサービス・介護施設向けライセンス（一括アカウント） | ★☆☆ 将来 |
| **広告** | **原則なし**。体験を壊すため | ✗ |

### 無料で提供する範囲（信頼のため）

- 毎日1宝箱（1〜3問）
- 基本的なしおりコレクション
- アルバム見返し（直近30日）
- 匿名共感の閲覧

### プレミアムで追加

- アルバム無制限・PDF出力
- ペア・家族モード
- 季節イベントの全問いアーカイブ
- カスタム問いの作成

## 9. MVPで作るべき機能

**目標**：6〜8週間で「毎日5分の思い出しあそび」が回る最小版

### 必須（MVP）

| ID | 機能 | 説明 |
|---|---|---|
| M-01 | 認証 | メール or 匿名ゲスト（後からアカウント連携） |
| M-02 | 今日の宝箱 | 1日1回、問い1〜2問を配信 |
| M-03 | 問いカタログ | 上記30問から日替わりローテ（重複防止） |
| M-04 | 回答入力 | テキスト（200字程度） |
| M-05 | 振り返り | 回答表示＋キャラ一言＋しおり獲得 |

## 10. 成功指標（脳トレ指標ではない）

| 指標 | 意味 |
|---|---|
| 週3日以上開封率 | 習慣になっているか |
| 平均セッション時間 3〜7分 | 5分設計が機能しているか |
| しおり累計枚数 | コレクションの蓄積 |
| 振り返り画面の滞在時間 | 自分の言葉を読んでいるか |
| オンボーディング完了率 | 世界観に入れたか |
| 「思い出せなかった」回答率 | 押し付けがないか（一定あってよい） |

**追わない指標**：正答率、スコア向上、ランキング順位
